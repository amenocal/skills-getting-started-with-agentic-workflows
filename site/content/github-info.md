- **Start Copilot cloud agent tasks via the REST API**: Developers can now trigger Copilot cloud agent sessions programmatically through the REST API, enabling automation and CI/CD integration. *(GitHub Changelog, May 13 2026)*

- **Copilot code review comment experience improved**: Copilot's inline code review comments are now easier to read and act on, streamlining the PR review loop. *(GitHub Changelog, May 12 2026)*

- **GitHub Copilot individual plans updated with flex allotments and a new Max plan**: Starting June 1, Pro and Pro+ plans gain flexible usage allotments and a new Max tier, giving individual developers more control over their Copilot usage. *(GitHub Blog, May 12 2026)*
- **GitHub Weekly Comment Sync agentic workflow**: A community-contributed workflow that automatically finds stale code comments or README snippets and opens a draft PR with text-only sync updates on a weekly schedule. *(Awesome Copilot Workflows)*
