- **Copilot CLI agent comes to JetBrains IDEs**: JetBrains users now get the Copilot CLI agent plus a unified sessions view showing live status for running and queued sessions. — [GitHub Changelog](https://github.blog/changelog/2026-05-13-introducing-copilot-cli-agent-and-unified-sessions-view-in-github-copilot-for-jetbrains-ides)

- **New GitHub Copilot individual plans (effective June 1)**: Pro and Pro+ gain flex allotments and a new Max plan is launching, giving individual developers more flexible access to Copilot models. — [GitHub Blog](https://github.blog/news-insights/company-news/github-copilot-individual-plans-introducing-flex-allotments-in-pro-and-pro-and-a-new-max-plan/)
- **GitHub Issues navigation now feels instant**: The Issues team shipped client-side caching, smart prefetching, and service workers so navigating between issues is near-instant with no full-page reloads. — [GitHub Blog](https://github.blog/engineering/architecture-optimization/from-latency-to-instant-modernizing-github-issues-navigation-performance/)
- **Weekly Comment Sync agentic workflow**: A community workflow automatically finds stale code comments or README snippets, makes text-only sync updates, and opens a draft PR — a practical pattern for keeping docs fresh. — [Awesome Copilot Workflows](https://awesome-copilot.github.com/workflows/)
- **Relevance Check slash command workflow**: A community workflow lets maintainers run `/relevance-check` on any issue or PR to get an AI evaluation of whether it is still worth pursuing. — [Awesome Copilot Workflows](https://awesome-copilot.github.com/workflows/)
